# 🚀 CareerVerse AI

> **Experience your dream career before choosing it.**

CareerVerse AI is an AI-powered career exploration platform that helps students experience different careers before making important academic and professional decisions. Instead of reading job descriptions, users interact with realistic AI simulations, complete hands-on tasks, receive personalized feedback, and explore career paths through immersive experiences.

---

## 🌟 Features

- 🤖 AI-powered career simulations
- 💼 Explore multiple career paths
- 🎯 Personalized career recommendations
- 🧠 Interactive real-world tasks and challenges
- 📊 Skill assessment and progress tracking
- 🏆 Gamified learning experience
- 📈 Career roadmap generation
- 🌍 Modern responsive UI
- 🔐 Secure authentication
- ⚡ Fast performance with Next.js

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Nehal390/careerverse.git
```

Move into the project

```bash
cd careerverse
```

Install dependencies

```bash
npm install
```

Create environment variables

```bash
cp .env.example .env.local
```

Run the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
